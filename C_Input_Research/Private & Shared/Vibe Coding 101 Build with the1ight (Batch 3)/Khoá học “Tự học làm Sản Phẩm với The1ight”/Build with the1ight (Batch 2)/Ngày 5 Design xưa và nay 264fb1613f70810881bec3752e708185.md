### Mục tiêu

- Hiểu sự khác biệt giữa quy trình vẽ **prototype truyền thống** (vẽ tay => giấy => Figma) và **prototype hiện đại** (prompt,AI tool)
- Biết cách tư duy theo user journey và vẽ user flow cho các sản phẩm.
- Biết rằng có sự tồn tại của Canva Code và Figma Make.
